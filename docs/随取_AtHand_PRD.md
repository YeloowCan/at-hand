# 随取 AtHand - 产品需求文档（PRD）

## 一、产品概述

### 1.1 产品名称

随取 AtHand

### 1.2 产品定位

一款**离线优先 + 高度加密 +
极速访问**的个人重要信息管理工具，帮助用户在关键时刻快速获取敏感信息。

### 1.3 核心价值

-   ⚡ 秒级获取（3秒内找到信息）
-   🔒 本地加密（数据不上传云端）
-   📋 一键复制（提升填写效率）

------------------------------------------------------------------------

## 二、用户痛点

-   相册查找效率低
-   信息分散在多个App
-   担心隐私泄露
-   重复输入成本高

------------------------------------------------------------------------

## 三、目标用户

-   城市年轻用户（20--40岁）
-   高频处理个人事务
-   注重效率与隐私

------------------------------------------------------------------------

## 四、核心功能

### 4.1 离线优先

-   数据仅存本地
-   无账号体系

### 4.2 数据安全

-   AES-256 加密
-   PIN / 指纹 / FaceID 解锁
-   剪贴板自动清除（可选）

### 4.3 快捷检索

-   模糊搜索（标题/内容）
-   分类（证件/合同/卡片）
-   最近使用排序

### 4.4 一键复制（核心功能）

-   支持身份证号、地址、卡号等
-   点击即复制
-   复制反馈提示

### 4.5 文档管理

-   文本 / 图片 / PDF
-   相册导入 / 拍照扫描
-   可扩展 OCR

### 4.6 分类与标签

-   默认分类 + 自定义分类
-   标签系统

### 4.7 快速录入

-   手动输入
-   图片导入
-   扫描录入

------------------------------------------------------------------------

## 五、信息架构

-   首页（搜索 + 最近使用 + 分类）
-   分类页
-   详情页
-   新建页
-   设置页

------------------------------------------------------------------------

## 六、交互设计原则

-   极简（3步完成操作）
-   高效（优先展示常用）
-   安全（强化隐私感）

------------------------------------------------------------------------

## 七、非功能需求

### 性能

-   启动 \< 1s
-   搜索 \< 200ms

### 安全

-   本地加密存储
-   防截屏（可选）

### 平台

-   iOS / Android / Web(PWA)

------------------------------------------------------------------------

## 八、技术方案

-   前端：React Native / Flutter
-   存储：SQLite / IndexedDB
-   加密：WebCrypto / Native Crypto

------------------------------------------------------------------------

## 九、MVP范围

### 必做

-   本地存储
-   加密
-   搜索
-   一键复制
-   图片上传

### 可延期

-   OCR
-   云备份
-   多设备同步

------------------------------------------------------------------------

## 十、风险与方案

-   丢失设备 → 提供本地备份
-   忘记密码 → 不可恢复（强调安全）
-   误删 → 回收站机制

------------------------------------------------------------------------

## 十一、未来规划

-   加密云备份
-   小组件
-   智能识别证件
-   自动分类

------------------------------------------------------------------------

## 十二、产品核心一句话

> 随取 AtHand：让重要信息，随时可得。
